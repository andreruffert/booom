# BOOOM!

> A simple CLI to scaffold some files or folder structures to kickstart new projects.

## Install

```
$ npm install --global booom
```

## Usage

```
booom --help
  
  Usage
    $ booom [url]

  Examples
    $ booom https://github.com/andreruffert/booom-initial-standard/archive/master.zip
```
